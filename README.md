🌾 Análise de Produtividade Agrícola e Modelagem de Dados com AWS
📌 Descrição do Projeto

Este projeto tem como objetivo analisar dados agrícolas e prever o rendimento de safras utilizando técnicas de Ciência de Dados e Modelagem Preditiva. Além disso, apresenta uma proposta de infraestrutura em nuvem utilizando AWS, incluindo estimativa de custos e arquitetura recomendada.

O projeto integra:

Análise Exploratória de Dados (EDA)

Modelagem supervisionada com regressão

Comparação de custos na AWS

Geração automatizada de gráficos e relatórios
